Photoshop

Photoshop se utiliza principalmente para la edición y manipulación de imágenes. Con esta herramienta, se pueden retocar fotos, crear composiciones artísticas, modificar el color y la iluminación de una imagen, realizar montajes, entre muchas otras funciones. Es ampliamente utilizado por fotógrafos, diseñadores gráficos, ilustradores y artistas digitales para dar vida a sus creativas ideas y producir trabajos visualmente impactantes.

Premier Pro

Adobe Premiere Pro es un software de edición de video profesional ampliamente utilizado en la industria del cine. Ofrece una variedad de herramientas de edición, efectos visuales, transiciones. Con funciones avanzadas como la edición en resoluciones 4K, audio de alta calidad y colaboración en la nube, Premiere Pro es una herramienta versátil para la creación de contenido visual impresionante y de alta calidad.

Diseño Web

El diseño web es la creación y disposición visual de sitios web. Implica combinar elementos como colores, tipografía, imágenes y diseño de página para lograr una experiencia atractiva y funcional para los usuarios. Los diseñadores web utilizan herramientas como HTML, CSS y JavaScript, Bootstrap para construir sitios que sean atractivos, funcionales, responsivos y optimizados.
